Unicodeでアイコン表示

ちょっとアイコン欲しいのにfontawesomeとかセットアップするのがめんどうなので、フォントのまま使えないか調査。

早速使ってみる

✅ &#9989;
❎ &#10062;
✎ &#9998;
♻ &#9851;
⌘ &#8984;
✐ &#10000;
➠ &#10144;
✕ &#10005;
❗ &#10071;
✔ &#10004;
➲ &#10162;

注意事項

  • フォントがインストールされていないとそもそも使えない
  • CSSのcontentプロパティには&#xxxx;形式が使えない
