Javascript

JavaScript Ninjaの極意★★★★★
サブタイトルは『ライブラリ開発のための知識とコーディング』。 jQueryの作者ジョン・レシグ氏による本。 Javascript忍者になるための技の数々が、とてもわかりやすく解説されています。 関数、プロトタイプ、正規表現に多くの章が割かれており、ものすごく勉強になります。 脱初心者にとりあえず一冊なら、これがオススメ。
JavaScript 第6版★★★★
Javascriptといえばの、定番オライリーのサイ本。 840ページの厚みでひたすらJavascriptを解説。 これでも原著を収録しきれず、『JavaScriptリファレンス 第6版』(392ページ)に分かれています。 知識補強のためにあってもいいかな。 内容も質量も重たいので電子書籍版をオススメします。
Effective JavaScript★★★
サブタイトルは『JavaScriptを使うときに知っておきたい68の冴えたやり方』。 『JavaScript Ninjaの極意』と内容がかぶります。 文体がつまらないので眠くなります。
JavaScript: The Good Parts★★
サブタイトルは『「良いパーツ』によるベストプラクティス』。 アンチ派も多く出版も古いため、参考程度。

HTML5/CSS3モダンコーディング★★★★
サブタイトルは「フロントエンドエンジニアが教える3つの本格レイアウト スタンダード・グリッド・シングルページレイアウトの作り方」。 とてもわかりやすい。 文字の間にわずか1pxのスキマを入れることでメニューが見やすくなるとか、こまかい気配りがすばらしい。 bootstrapなどのcssフレームワークを使わずにCSSを生で書くためのtipsが満載なのでオススメ。
現場のプロが教えるWebデザイン＆レイアウトの最新常識★★★
Amazonでの評価はかなり高いが。。。 あまり記憶に残ってないし読み返す気もなかった。 でも上記の「HTML5/CSS3モダンコーディング」を読んでから読み返したら、内容がすっと入ってきておもしろい。 CSSやJSコーディングスキルを上げるための書籍ではなく、こんなこともできますよーっていうインデックス作りの読み物としてどうぞ。
