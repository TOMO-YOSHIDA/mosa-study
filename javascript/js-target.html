JS学習のターゲットバージョン

JSのバージョン

JavascriptにはES3、ES5、ES6などいくつかバージョンが存在します。 最近ではalterJSといって、別言語で記述してJavascriptに変化する技術があります。 有名どころではCoffeeScript、TypeScript、Babelなど。
JSの歴史についてはけっこう長くなるのではしょります。
最近のブラウザは、モバイルを含めECMAScript Version 5(ES5)がだいたい使えます。 「今使えるJavascript」を考えて、最大公約数的にES5を基準にすすめます。

最低限必要な知識

  • if, else, else if
  • for, while
  • switch,case,default
  • 配列の概念

学習の進め方

chromeでページを開いたら、開発者ツールを開いて画面の左右に分けてください。 コンソールにソースをコピー＆ペーストしながら見ていくとわかりやすいと思います。
