Javascriptの型

string

文字列を表す。長さには特に制限はなく、メモリいっぱいまで使えるっぽい。

number

  • いわゆる倍精度浮動小数点数(IEEE754標準の64ビット浮動小数点数形式)
  • 範囲は±1.7976931348623157×10308

function

  • 関数として使える(当然)
  • オブジェクトのプロパティの場合、メソッドとして使える
  • コンストラクタとして使える
  • 関数にもプロパティを定義できる

object
