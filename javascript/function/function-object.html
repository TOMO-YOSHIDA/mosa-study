オブジェクトとして使う関数

関数はコンストラクタとして呼び出し、実体化することができます。 Javaでいうクラスの様な扱いになります。 関数でありながら、オブジェクトです。

Javascriptにおけるfunction

関数は、newを付けて呼び出すことで実体化することができます。
この例では、関数自体はインスタンスの外からも参照できる変数を保持するだけで、特に機能を持っているわけではありません。
関数がインスタンス化できること、プロパティを持てることを示しました。
変数のカプセル化については「クロージャ」で、メソッドについては「プロトタイプ」で解説します。

newをつけずに呼び出す

コンストラクタ関数は、newをつければ期待通りにコンストラクタとして動作します。 ただし、newをつけずにコールした場合、関数としてのコールになります。
それがどんな悪さをするのかは、「this」で解説します。
